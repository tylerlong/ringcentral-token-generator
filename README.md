# ringcentral-token-generator

RingCentral token generator


![RingCentral Token Generator](screenshot.png)


### [Download](https://github.com/tylerlong/ringcentral-token-generator/releases)


## Setup

```
yarn install
yarn run upgrade
```


## Development

```
yarn build:watch
yarn start
```


## Pack

```
yarn dist:dev
```


## Distribution

```
yarn dist
```
